# Starke ST - Suporte Técnico Profissional

![Starke ST Logo](assets/img/favicon.svg)

## 📋 Sobre o Projeto

A Starke ST é uma empresa especializada em suporte técnico e soluções tecnológicas, oferecendo serviços profissionais em hardware e software para empresas e pessoas físicas. Este repositório contém o código-fonte do site institucional da empresa.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.3
- Bootstrap Icons
- AOS (Animate On Scroll)
- Swiper.js
- GLightbox
- Isotope
- PureCounter

## ✨ Funcionalidades

- Design responsivo e moderno
- Animações suaves
- Carrossel de clientes
- Galeria de portfólio
- Formulário de contato
- Seções interativas
- Otimizado para SEO
- Suporte a PWA (Progressive Web App)

## 🛠️ Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/starke-st.git
```

2. Navegue até o diretório do projeto:
```bash
cd starke-st
```

3. Abra o arquivo `index.html` em seu navegador ou utilize um servidor local.
